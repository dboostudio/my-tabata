# 📋 요구사항 명세서 — 타바타 타이머

> 작성자: PM | 버전: 1.0

---

## 1. 제품 개요

**제품명**: TabataGo
**한 줄 설명**: 가장 심플하고 아름다운 타바타 타이머 — 운동에만 집중하세요
**플랫폼**: PWA (웹앱, 모바일/데스크탑 모두 지원, 서버 없음)

---

## 2. 타겟 사용자

### 페르소나 A — 홈트 입문자 (주 타겟)
- 20~35세, 헬스장 대신 집에서 운동
- 타바타 프로토콜을 알지만 타이머 앱을 찾는 중
- 간단하고 직관적인 UI를 원함

### 페르소나 B — 크로스핏/HIIT 마니아 (프리미엄 타겟)
- 25~40세, 다양한 인터벌 트레이닝을 즐김
- 커스텀 타이머 설정, 워크아웃 프리셋 필요
- 기록 관리를 통해 동기부여 받음

---

## 3. 기능 정의 (Free / Pro 티어)

### Free 티어

| 기능 | 설명 |
|------|------|
| 기본 타바타 타이머 | 20초 운동 / 10초 휴식 / 8라운드 고정 |
| 시각적 카운트다운 | 원형 프로그레스 링 + 대형 숫자 |
| 페이즈 표시 | WORK / REST / 라운드 진행 상황 |
| 시작/일시정지/리셋 | 기본 컨트롤 |
| 비프음 알림 | Web Audio API 기반 소리 (운동 시작/휴식 시작/완료) |
| PWA 설치 | 홈 화면에 앱으로 추가 가능 |

### Pro 티어 ($4.99 일회성 결제)

| 기능 | 설명 |
|------|------|
| 커스텀 인터벌 | 운동 시간, 휴식 시간, 라운드 수 자유 설정 |
| 워크아웃 프리셋 | HIIT, 크로스핏, 요가, 필라테스 등 10개+ 프리셋 |
| 음성 안내 | "운동 시작!", "휴식!", "마지막 라운드!" 음성 (Web Speech API) |
| 운동 기록 | 완료한 운동 히스토리 (LocalStorage) |
| 다크/라이트 테마 | 3가지 컬러 테마 |
| 준비 카운트다운 | 운동 시작 전 3-2-1 카운트 |
| 화면 켜짐 유지 | Wake Lock API로 운동 중 화면 꺼짐 방지 |

## 4. 사용자 스토리

```
[Free] 사용자로서, 앱을 열면 바로 타바타 타이머를 시작할 수 있다.
[Free] 사용자로서, 운동/휴식 전환 시 소리로 알림을 받는다.
[Free] 사용자로서, 현재 라운드와 남은 시간을 한눈에 확인한다.
[Pro]  사용자로서, 인터벌 시간을 내 루틴에 맞게 조정한다.
[Pro]  사용자로서, 저장된 프리셋으로 빠르게 워크아웃을 시작한다.
[Pro]  사용자로서, 음성 안내로 화면을 보지 않고도 운동한다.
[Pro]  사용자로서, 지난 운동 기록을 확인하며 동기부여를 받는다.
```

## 5. 비기능 요구사항

| 항목 | 요구사항 |
|------|---------|
| 성능 | 첫 로드 3초 이내, 타이머 정확도 ±50ms 이내 |
| 오프라인 | 설치 후 인터넷 없이 완전 동작 |
| 반응형 | 모바일(320px) ~ 데스크탑(1920px) 모두 지원 |
| 접근성 | WCAG 2.1 AA 수준 |
| 서버 의존성 | 없음 (완전 클라이언트 사이드) |

## 6. MoSCoW 우선순위

**Must Have**: 기본 타이머, 소리, 시각적 UI, PWA
**Should Have**: 커스텀 인터벌, 프리셋, 프리미엄 결제 연동
**Could Have**: 음성 안내, 운동 기록, 테마
**Won't Have (v1)**: 소셜 공유, 클라우드 동기화, 멀티플레이어
