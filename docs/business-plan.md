## 수익 모델: Freemium + 일회성 결제

### 가격 전략
| 티어 | 가격 | 결제 방식 |
|------|------|---------|
| Free | $0 | 없음 |
| Pro | **$4.99** (일회성) | Gumroad 또는 Stripe Payment Link |

**$4.99 선정 이유**: 충동구매 임계값 이하, 앱스토어 유료 앱과 동일 수준, 경쟁 타이머 앱 평균가

---

## 결제 연동 (서버 없음)

### 방법: Gumroad 링크 방식

```
1. Gumroad에서 "$4.99 TabataGo Pro" 상품 등록
2. 앱 내 "Pro로 업그레이드" 버튼 → Gumroad 결제 페이지 (외부 링크)
3. 결제 완료 후 Gumroad이 설정한 redirect URL로 이동:
   https://yourapp.vercel.app?unlocked=tabatago_pro_2024
4. 앱이 URL 파라미터를 감지 → localStorage에 Pro 상태 저장
5. 이후 Pro 기능 잠금 해제
```

**보안 수준**: 낮음 (클라이언트 사이드) — 인디 앱 수준으로 충분
**대안**: Lemon Squeezy License Key API (서버 없이 검증 가능한 방식)
